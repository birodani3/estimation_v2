# What is it?
A tool for easy, fast, and fun ticket estimating.

# Running locally
Downloading dependencies
* npm i
* cd server
* npm i

Starting the server:
* cd server
* node main.js

Starting the client:
* npm run watch
* Serve index.html with any simple webserver
    * npm i -g live-server
    * live-server
